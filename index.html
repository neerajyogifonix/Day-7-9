Debouncing

Debouncing ensures that a function (like searching) only runs after the user stops typing for a certain time. This prevents unnecessary calls, Load on server and improves performance.

Throttling

Throttling ensures that a function (like button clicks) runs at most once in a given time period, even if the event is triggered many times.

Currying

Currying transforms a function with multiple arguments into a sequence of single-argument functions. This helps with reusability and creating specialized functions.
